# EC421 Problem Set 7

This folder contains my solution to Problem Set 7 from EC421 (Econometrics course). The analysis involves regression modeling, correlation analysis, and descriptive statistics on financial/economic data.

## Files
- `EC421 ps7.ipynb`: Jupyter notebook with the complete analysis, including data loading, modeling, and visualizations.
- `correlation_matrix.csv`: Correlation matrix of variables.
- `descriptive_stats.csv`: Summary statistics.
- `pass_through_dataset.csv`: Processed dataset.
- `regression_compact_table.csv`: Regression results summary.
- `regression_summary_table.csv`: Detailed regression output.

## Key Topics
- Linear regression
- Model diagnostics
- Data visualization

Run the notebook in Jupyter to reproduce the results.
