# IMC Prosperity Trading Competition

This folder contains my work from the IMC Prosperity Trading Competition, focusing on the tutorial round. It includes trading strategies, data analysis, and backtesting tools.

## Files
- `Tutorial round/`: Contains strategy implementations (trader_1.py to trader_4.py), setup guide, and data.
- `imc-prosperity-4-backtester/`: Backtesting framework for simulating trades.
