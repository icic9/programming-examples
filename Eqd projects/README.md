# Eqd Projects

This folder contains financial modeling and trading tools developed in Python.

## Files
- `D1_Basket_Trader_Hedge_Monitor.ipynb`: Notebook for monitoring basket trader hedges, including risk analysis.
- `Option_Price_Engine.ipynb`: Comprehensive options pricing engine implementing Black-Scholes model with Greeks calculation, OOP design, and scenario analysis.

## Key Features
- Vectorized Black-Scholes pricing
- Greeks computation (Delta, Gamma, Vega)
- Position and book-level risk management
- Scenario shock analysis and PnL attribution

Run the notebooks in Jupyter. The Option_Price_Engine includes a complete trading book simulation.
